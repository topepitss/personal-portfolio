# Joshua M. Lugay Portfolio

A modern personal portfolio website built with React, Vite, JavaScript, and CSS.

## Run Locally

```bash
npm install
npm run dev
```

## Update Your Information Without Editing Layout Code

- Personal details, contact info, About text, and Education: `src/data/profile.js`
- Projects: `src/data/projects.js`
- Skills: `src/data/skills.js`
- Experience: `src/data/experience.js`
- Certifications: `src/data/certifications.js`
- CV file: replace `public/cv.pdf`
- Profile photo: replace `public/profile.jpg`
- Project gallery screenshots: replace `src/assets/project-images/tracker-1.png` through `tracker-5.png`
